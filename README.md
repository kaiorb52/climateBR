# climateBR

**climateBR** provides tools to download, process, and analyze historical meteorological data from the Brazilian National Institute of Meteorology (INMET).

## Installation

The package is not yet available on CRAN. You can install the development version from GitHub:

```r
# install.packages("remotes")

remotes::install_github("kaiorb52/climateBR")
```
